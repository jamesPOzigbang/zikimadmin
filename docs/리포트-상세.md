# 리포트 상세 페이지 PRD

## 📄 페이지 정보

- **파일명**: `report-detail.html`
- **페이지명**: 리포트 세부내용
- **URL 경로**: `/report-detail.html?id={리포트번호}`

---

## 1. 개요

### 1.1 페이지 목적

리포트 상세 페이지는 발급된 리포트의 모든 상세 정보를 조회할 수 있는 페이지입니다. 관리자는 이 페이지를 통해 리포트의 매물 요약 정보, 진단 항목 상세, 범죄/치안 정보, 생활/편의 정보를 종합적으로 확인할 수 있습니다.

### 1.2 주요 기능

- 리포트 기본 정보 조회
- 매물 요약 정보 표시
- 진단 항목 상세 조회 (21개 체크리스트 항목)
- 범죄/치안 정보 조회
- 생활/편의 정보 조회
- 등기부등본 PDF 다운로드

### 1.3 사용자 시나리오

관리자가 리포트 발급 이력 페이지에서 특정 리포트를 클릭하면, 해당 리포트의 상세 페이지로 이동합니다. 이 페이지에서 리포트의 모든 정보를 확인하고, 필요시 PDF 문서를 다운로드할 수 있습니다.

## 2. 정책

### 2.1 접근 권한

- **관리자 권한**: 리포트 조회 권한이 있는 관리자만 접근 가능
- **인증**: 관리자 인증이 완료된 사용자만 접근 가능
- **리포트 접근**: 리포트 번호를 URL 파라미터로 전달받아 해당 리포트만 조회 가능

### 2.2 데이터 표시 정책

#### 2.2.1 리포트 정보 표시
- **리포트 ID**: 페이지 헤더에 리포트 ID와 도로명 주소 표시
- **매물 요약**: 리포트의 핵심 정보를 카드 형태로 표시
- **진단 항목**: 21개 체크리스트 항목을 그리드 형태로 표시
- **주소 형식**: 부동산 주소는 도로명 주소로 표시

#### 2.2.2 상태 표시
- **양호**: 초록색 배지 또는 표시
- **확인 필요**: 주황색 배지 또는 표시

### 2.3 정보 섹션 정책

리포트 상세 페이지는 다음 4개의 주요 섹션으로 구성됩니다:
1. **매물 요약**: 리포트의 기본 정보 및 시세 정보
2. **진단 항목 상세**: 21개 체크리스트 항목의 상세 정보
3. **범죄/치안**: 해당 지역의 범죄 및 치안 관련 정보
4. **생활/편의**: 해당 지역의 생활 편의 시설 정보

### 2.4 PDF 다운로드 정책

- **등기부등본 PDF**: 건물 등기부등본과 토지 등기부등본 PDF 다운로드 제공
- **접근 권한**: 관리자만 다운로드 가능
- **파일 형식**: PDF 형식

### 2.5 데이터 범위 정책

#### 2.5.1 범죄/치안 정보
- **반경 기준**: 반경 500m 기준으로 시설 정보 제공
- **데이터 출처**: 경찰청 등 공식 기관 데이터 활용
- **비교 대상**: 인근 지역과의 비교 정보 제공

#### 2.5.2 생활/편의 정보
- **반경 기준**: 반경 500m 기준으로 시설 정보 제공
- **비교 기능**: 사용자의 기존 거주 지역과 비교 가능

## 3. 기능

### 3.1 리포트 기본 정보 표시

#### 3.1.1 페이지 헤더
- **리포트 ID**: 리포트 고유 식별자 표시
- **주소**: 부동산 도로명 주소 표시
- **뒤로가기 버튼**: 리포트 발급 이력 페이지로 돌아가기

### 3.2 매물 요약 섹션

#### 3.2.1 기본 정보
다음 정보를 그리드 형태로 표시합니다:
- **발급 일시**: 리포트 발급 일시
- **계약 기간**: 임대차 계약 기간, 매매일 경우엔 "-"로 표시
- **보증금**: 보증금 금액, 매매일 경우 '매매금'으로 표시
- **시세 추정가**: 부동산 시세 추정 금액
- **등기부등본 융자**: 등기부등본에 기록된 융자 금액
- **보증 가입 가능 여유금액**: 보증 가입 가능한 여유 금액 (강조 표시)
- **최우선 변제금**: 최우선 변제 가능 금액, 매매 경우엔 "-"로 표시

#### 3.2.2 PDF 다운로드
- **건물 등기부등본 PDF**: 건물 등기부등본 PDF 다운로드 버튼
- **토지 등기부등본 PDF**: 토지 등기부등본 PDF 다운로드 버튼

### 3.3 진단 항목 상세 섹션

#### 3.3.1 체크리스트 항목 표시
21개의 체크리스트 항목을 카드 형태로 표시합니다:
- 위반건축물
- 대지권
- 토지별도등기
- 가등기
- 압류/가압류
- 처분금지가처분
- 신탁
- 경매개시결정
- 근저당권
- 임차권등기명령
- 전세권
- 민간임대주택등록 여부
- 등기사항 변경예정
- 건물,토지 소유자 일치
- 임대사업자 등록
- 보증금미반환 이력
- 기존채무금액
- 여유금액
- 보증보험 예비심사
- 보증금 대출 예비 심사
- 최우선 변제권

#### 3.3.2 항목별 상세 정보
각 항목 카드에는 다음 정보가 포함됩니다:
- **항목명**: 체크리스트 항목 이름
- **상태 표시**: 양호 또는 확인 필요 배지
- **안내**: 항목에 대한 안내 내용
- **설명**: 항목에 대한 설명 내용
- **특약 사항**: 확인 필요 상태일 경우 특약 사항 안내 (주황색 강조)

### 3.4 범죄/치안 섹션

#### 3.4.1 방범 시설 분포
- **지도 표시**: 반경 500m 내 방범 시설 위치 지도
- **통계 정보**:
  - 방범 CCTV 개수
  - 안전비상벨 개수
  - 경찰서/파출소 개수
  - 소방서 개수
  - 24시 편의점 개수

#### 3.4.2 안전 귀갓길 체크
- **경로 표시**: 최단거리 도보 기준 안전 귀갓길 경로 지도
- **경로 내 시설**:
  - 경로 내 방범 CCTV 개수
  - 경로 내 안전비상벨 개수

#### 3.4.3 동네 유흥업소 수 비교
- **비교 차트**: 반경 500m 내 유흥업소 수를 막대 그래프로 표시
- **비교 대상**: 인근 동네와 비교 (예: 망원동, 서교동, 성산동)
- **현재 동네 강조**: 현재 매물이 위치한 동네는 강조 표시

#### 3.4.4 지난해 범죄 발생 수 비교
- **범죄 통계**: 지난해 범죄 발생 건수를 막대 그래프로 표시
- **비교 대상**: 인근 구 단위 비교 (예: 마포구, 은평구, 서대문구)
- **데이터 출처**: 경찰청 공식 데이터

### 3.5 생활/편의 섹션

#### 3.5.1 편의 시설
- **지도 표시**: 반경 500m 내 편의 시설 위치 지도
- **시설 통계**:
  - 24시 편의점 개수
  - 병원/약국 개수
  - 외식/카페 개수
  - 교육/학원가 개수
  - 대형마트 개수
  - 대중교통 개수

#### 3.5.2 내 동네와 비교하기
- **레이더 차트**: 6가지 편의 시설 항목을 레이더 차트로 표시
- **비교 항목**:
  - 편의점
  - 대형마트
  - 대중교통
  - 교육/학원가
  - 외식/카페
  - 병원/약국
- **비교 대상**: 사용자의 기존 거주 지역과 비교 (예: 목동)
- **범례**: 현재 동네와 비교 대상 동네를 구분하여 표시

### 3.6 네비게이션 기능

#### 3.6.1 뒤로가기
- **목록으로 돌아가기**: 리포트 발급 이력 페이지로 이동
- **브레드크럼**: 홈 > 리포트 발급 이력 > 리포트 세부내용

## 4. 유저스토리

### 4.1 리포트 상세 정보 조회

**As a** 관리자  
**I want to** 리포트의 모든 상세 정보를 한 페이지에서 확인할 수 있어야  
**So that** 리포트의 전체적인 내용을 파악하고 사용자에게 제공할 수 있다

**Acceptance Criteria:**
- 리포트 ID와 도로명 주소가 페이지 헤더에 표시된다
- 매물 요약, 진단 항목, 범죄/치안, 생활/편의 섹션이 모두 표시된다
- 각 섹션의 정보가 명확하게 구분되어 표시된다

### 4.2 매물 요약 정보 확인

**As a** 관리자  
**I want to** 리포트의 핵심 정보(보증금, 시세, 여유금액 등)를 빠르게 확인할 수 있어야  
**So that** 리포트의 주요 내용을 즉시 파악할 수 있다

**Acceptance Criteria:**
- 발급 일시, 계약 기간, 보증금, 시세 추정가 등이 표시된다
- 보증 가입 가능 여유금액이 강조되어 표시된다
- 정보가 그리드 형태로 정리되어 가독성이 좋다

### 4.3 등기부등본 PDF 다운로드

**As a** 관리자  
**I want to** 등기부등본 PDF를 다운로드할 수 있어야  
**So that** 리포트와 함께 관련 문서를 확인하거나 보관할 수 있다

**Acceptance Criteria:**
- 건물 등기부등본 PDF 다운로드 버튼이 제공된다
- 토지 등기부등본 PDF 다운로드 버튼이 제공된다
- 버튼 클릭 시 PDF 파일이 다운로드된다

### 4.4 진단 항목 상세 확인

**As a** 관리자  
**I want to** 각 체크리스트 항목의 상세 정보를 확인할 수 있어야  
**So that** 리포트의 진단 내용을 정확히 파악할 수 있다

**Acceptance Criteria:**
- 21개의 체크리스트 항목이 모두 표시된다
- 각 항목의 상태(양호/확인 필요)가 명확하게 표시된다
- 안내, 설명, 특약 사항이 표시된다
- 확인 필요 항목의 특약 사항이 강조되어 표시된다

### 4.5 범죄/치안 정보 확인

**As a** 관리자  
**I want to** 해당 지역의 범죄 및 치안 정보를 확인할 수 있어야  
**So that** 리포트에 포함된 범죄/치안 정보의 정확성을 검증할 수 있다

**Acceptance Criteria:**
- 방범 시설 분포 지도와 통계가 표시된다
- 안전 귀갓길 경로와 경로 내 시설 정보가 표시된다
- 동네 유흥업소 수 비교 차트가 표시된다
- 지난해 범죄 발생 수 비교 차트가 표시된다

### 4.6 생활/편의 정보 확인

**As a** 관리자  
**I want to** 해당 지역의 생활 편의 시설 정보를 확인할 수 있어야  
**So that** 리포트에 포함된 생활/편의 정보의 정확성을 검증할 수 있다

**Acceptance Criteria:**
- 편의 시설 지도와 통계가 표시된다
- 내 동네와 비교하기 레이더 차트가 표시된다
- 비교 대상 동네가 명확하게 구분되어 표시된다

### 4.7 리포트 목록으로 돌아가기

**As a** 관리자  
**I want to** 리포트 상세 페이지에서 목록 페이지로 쉽게 돌아갈 수 있어야  
**So that** 여러 리포트를 효율적으로 조회할 수 있다

**Acceptance Criteria:**
- "목록으로 돌아가기" 버튼이 제공된다
- 버튼 클릭 시 리포트 발급 이력 페이지로 이동한다
- 브레드크럼을 통해서도 이동할 수 있다
